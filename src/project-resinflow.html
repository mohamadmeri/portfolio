ResinFlow Analytics

Manufacturing Cost Optimization Platform

Python pandas NumPy Plotly ETL Pipeline Data Analytics

Project Overview

Led a consulting engagement for a plastics manufacturer, analyzing resin substitution events across multiple machines and production months to uncover cost and efficiency drivers.

Key Achievements

ETL Pipeline Development

Engineered a Python-based ETL pipeline using pandas, NumPy, and openpyxl that consolidated 8 messy Excel workbooks with thousands of records into a single, validated dataset.

Interactive Dashboards

Developed 7 interactive dashboards using Plotly with drill-down capabilities, deployed internally via Cloudflare Pages, providing leadership with real-time visibility into profitable vs. loss-making resin blends, machines, and parts.

Automated KPI Reports

Built a canonical data model and automated KPI reporting system tracking cost variance, substitution rates, and part/machine/material performance metrics.

Statistical Analysis

Performed comprehensive statistical analysis identifying key cost drivers, revealing that 92% of cost increases were concentrated in just 3 operational areas out of 50+.

Actionable Insights

Delivered a comprehensive analysis report with actionable recommendations, enabling the client to confirm that 79% of substitutions were cost-neutral or saving and to target loss-driving material swaps for corrective action.

Decision Support Tool

Empowered decision-makers with a secure internal decision-support tool, improving transparency, reducing waste, and enhancing profitability across production operations.

Technical Implementation

Data Engineering

Developed a robust ETL pipeline that automated the consolidation of disparate Excel workbooks. The pipeline included data validation, normalization, and quality checks to ensure data integrity. Used pandas for efficient data manipulation and openpyxl for Excel file handling.

Data Modeling

Created a canonical data model that standardized resin substitution events, machine operations, and material costs. This model served as the foundation for all downstream analytics and reporting.

Visualization & Analytics

Built interactive dashboards using Plotly that allowed stakeholders to drill down from high-level trends to specific machine-part-material combinations. Dashboards were deployed securely via Cloudflare Pages for internal access.

Automated Reporting

Implemented automated KPI calculation and report generation, eliminating manual processes and ensuring consistent, up-to-date insights for decision-makers.
